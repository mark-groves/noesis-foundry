## Summary

The project README defines Noesis as a local-first human-agent memory system.
It frames useful memory as a lifecycle rather than accumulation.

## Key Claims

- Noesis should help humans and agents carry understanding forward across time.
- Useful memory requires structure, review, context, and lifecycle.
- The system should transform sources into evidence, claims, synthesis, reviewed
  knowledge, and operational context.
- Old information should remain traceable while no longer guiding active work
  when it is stale or unreliable.
