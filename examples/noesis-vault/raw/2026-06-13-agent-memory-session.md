# Agent Memory Dogfood Session

During a Noesis project session, an agent used repo files, test output, and
review decisions as source material for future work. The useful memory was not
the raw terminal transcript. The useful memory was a source-backed handoff:
capture the project source, extract evidence, propose a narrow claim, review
that claim, synthesize the result, promote reviewed knowledge, and build
operational context from only the reviewed knowledge.

An older memory shortcut said agents could copy global summary snippets directly
into future prompts. That shortcut is stale because it bypasses local evidence
and makes it hard to identify what was reviewed.
