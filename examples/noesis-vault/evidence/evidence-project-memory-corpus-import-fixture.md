# Project Memory Corpus Import Fixture Evidence

## Evidence

The Codex session bundle fixture models local project artifacts with manifest
metadata, transcript content, provenance, evidence draft text, and a duplicate
artifact that should not become extra active context by accident.

## Source Basis

- [[source-project-memory-corpus-bundle-fixture]]

## Extraction Notes

This evidence supports preserving and reviewing imported project-session
material. It does not support treating captured artifacts as reviewed knowledge
without claim, review, and synthesis steps.

## Candidate Claims

- Project-session imports are useful as sources and evidence candidates, but
  active project memory should still be promoted through review.
