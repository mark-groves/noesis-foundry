# Stale Memory - Build Custom Obsidian Plugin First

## Previous Assumption

Noesis may need a custom Obsidian plugin before it can provide a useful human
review interface.

## Why It Is Superseded

The first-slice design can use Obsidian Properties, Bases, Canvas, templates,
and links over ordinary Markdown files. A custom plugin would add maintenance
surface before the lifecycle contract has been validated.
